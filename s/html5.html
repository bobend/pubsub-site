PUBSUB.IO

a query based message hub


				// connect from node
				var pubsub = require('pubsub.io').connect('hub.pubsub.io');
				// or the browser
				var pubsub = pubsub.connect('hub.pubsub.io');

				pubsub.subscribe({
					hello:{$any:['world','mundo','verden']}
				}, function(doc) {
					console.log(doc);
				});

				pubsub.publish({hello:'world'});

clients are currently available for node.js 

npm install pubsub.io

and browser side js 

<script src='http://pubsub.io/pubsub.io.js'></script>
